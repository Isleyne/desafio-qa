# Project Automation Tests

### Built With
* Cypress

### Requirements
* Node.js
* Chrome

### Install the Project Dependencies
```bash
Dentro da pasta: cypress
$ npm install
```

#### Run Tests

* Cypress console
```bash
   $ npm run cypress:open
```
* Headless mode
```bash
   $ npm run cypress:run
```
